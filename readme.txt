PassBy[ME] improves the security of your WordPress website. Enable two-factor authentication for your users now.

== Description ==

PassBy[ME] use two-factor authentication with the highest known security standard, which is PKI. Rather than relying on a password authentication, which can be phished or guessed, PassBy[ME] second factor service adds a second layer of security to your WordPress accounts. It protects your website from hacks and unauthorized login attempts.

PassBy[ME] is easy to setup and use. You do not need software installation, just sign up for PassBy[ME] service and install the plugin.

Protect your WordPress website with PassBy[ME] The next generation authentication solution.

== Installation ==
= From your WordPress dashboard =
1. Visit `Plugins > Add New`.
2. Search for `passbyme`. Find and Install `passbyme-two-factor-authentication`.
3. Activate the plugin from your Plugins page.

= From WordPress.org =
1. Download passbyme-two-factor-authentication.
2. Unzip and upload `passbyme-two-factor-authentication` directory to your `/wp-content/plugins/` directory.
3. Activate passbyme-two-factor-authentication from your Plugins page.

= Once Activated =
1. Select PassBy[ME] from the left menu and follow the instructions.
2. Once, you completed your setup. Set your users authentication in the "General" menu.
